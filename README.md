源码本质为火山源码, 通过火山平台翻译后,导出C++源码~

# 简介
```
1：内存占用小~ (120万文件占用150M~)
2：搜索速度快~ (120万文件搜索耗时0毫秒~ )
3：同时支持拼音+首拼+通配符混合搜索 比如: (*bcheng助?)
4：实时更新 (文件实时同步)
5：支持插件系统
```

# 使用方法：
> 激活搜索窗口：按下鼠标中键 或者 Alt + A
> 激活气泡菜单：按住鼠标右键并拖动鼠标 或者 按住Alt
